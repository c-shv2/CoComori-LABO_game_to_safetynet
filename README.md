# CoComori LABO - こころの守護者に出逢う旅

性格診断風のインタラクティブコンテンツです。質問に答えると、4系統 x 4タイプの合計16種類から「こころの守護者」キャラクターの結果が表示されます。

## 内容

- `index.html`: 診断ページ本体
- `result_*.png`: 結果画面で表示する通常画像
- `result_*_share.png`: 保存・シェア用画像
- `app.js`, `data.js`, `styles.css`: 分割管理用の補助ファイル

## 使い方

`index.html` をブラウザで開くと動きます。サーバーなしでも確認できます。

結果画面では以下が使えます。

- 画像を保存: 対応する `_share.png` 画像を保存
- LINEで送る: 友達向けの短い文言で共有
- Xでシェア: キャラ名とキーワードを含めて共有

## 公開するとき

GitHub Pagesなどで `https://...` のURLから開かれた場合、LINE/Xの共有文に診断URLが自動で追加されます。ローカルの `file://...` で開いている間は、PC内のパスが共有文に入らないようにしています。

## 画像ルール

通常表示用と保存用を分けています。

- 画面表示: `result_driver_2.png`
- 保存用: `result_driver_2_share.png`

16タイプすべてに `_share.png` が必要です。
