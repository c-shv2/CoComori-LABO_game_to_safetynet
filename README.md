# CoComori Guardian Diagnosis

> あなたを守ってきた子に出会う診断ゲーム

---

## このプロジェクトについて

このリポジトリは、CoComori LABOで制作している診断ゲーム
**「こころの守護者を探す旅」** の開発用リポジトリです。

質問に答えながら心の迷宮を進み、自分を守り続けてきた**16人の守護者**のうち、今の自分に近いキャラクターと出会う体験を目指しています。

一般的な性格診断のように人を分類することではなく、

**「これまで自分を守るために働いてきた心の反応」**

を、キャラクターとの出会いとして受け取れるように設計しています。

---

## コンセプト

「こころの守護者を探す旅」は、性格を当てるためだけの診断ではありません。

プレイヤーは、いくつかの場面で自分に近い反応を選びながら、心の奥にいる守護者へ近づいていきます。

診断結果は、長所や短所のラベルではなく、

**「あなたを傷つけないために、ずっと働いてきた守り方」**

として描かれます。

---

## この診断ゲームが目指すもの

- 防衛反応を否定しない
- 生きづらさを「欠点」ではなく「守る力」として捉える
- キャラクターとの出会いを通して自己理解を深める
- 心理学だけに閉じず、当事者研究・社会福祉・人権の視点も大切にする
- でもまずは、プレイヤーが安心して楽しめる診断ゲームとして成立させる

---

## CoComori LABOについて

CoComori LABOは、

**「当事者参加型・社会的ウェルビーイング実践研究所」**

として、生きづらさを抱える当事者の経験を、哲学・人権論・社会学・社会福祉などの視点から研究し、社会へ還元することを目的としています。

この診断ゲームは、その大きな活動の中から生まれた小さな実験の一つです。

研究や社会実装の入口である前に、まずは一つの体験として、

**「自分を守ってきた子に会えた」**

と思えるものを目指しています。

## 現在の実装

- 16タイプ診断
- 2段階の質問設計
- キャラクター結果画面
- 結果画像の表示
- 画像保存ボタン
- LINE共有ボタン
- X共有ボタン

結果画面では、通常表示用の画像と保存・共有用の画像を分けています。

- 画面表示: `result_driver_2.png`
- 保存用: `result_driver_2_share.png`

ローカルの `file://` で開いている間は、PC内のパスがLINE/Xの共有文に入らないようにしています。GitHub Pagesなどで `https://...` の公開URLから開かれた場合だけ、共有文に診断URLが自動で追加されます。
