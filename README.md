# CoComori LABO - こころの守護者を探す旅

> あなたを守ってきた子に出会う診断

---

## このプロジェクトについて

このリポジトリは、**CoComori LABO**で企画・研究・開発を進めているコンテンツの一つです。

CoComori LABOは、

**「当事者参加型・社会的ウェルビーイング実践研究所」**

として、生きづらさを抱える当事者の経験を、哲学・人権論・社会学・社会福祉などの視点から研究し、社会へ還元することを目的としています。

その研究の中から生まれた実践の一つが、

**「こころの守護者を探す旅」**

です。

---

## コンセプト

「こころの守護者を探す旅」は、一般的な性格診断ではありません。

プレイヤーが心の迷宮を旅し、自分を守り続けてきた**16人の守護者**と出会う体験を提供します。

診断結果は性格を分類するためではなく、

**「あなたを守るために生まれた心の守護者」**

との出会いとして描かれます。

---

## この診断が目指すもの

- 防衛反応を否定しない
- 生きづらさを「欠点」ではなく「守る力」として捉える
- キャラクターとの出会いを通して自己理解を深める
- 心理学だけでなく、当事者研究・社会福祉・人権の視点を取り入れる

---

## CoComori LABOについて

CoComori LABOでは、

- 当事者知の言語化と共有
- 質的研究による構造の可視化
- コミュニティでの知の循環
- 社会への橋渡し（アドボカシー）

を柱として活動しています。

本プロジェクトも、その研究成果を社会へ届けるための実践の一つです。

---

## 現在の実装

- 16タイプ診断
- 2段階の質問設計
- キャラクター結果画面
- 結果画像の表示
- 画像保存ボタン
- LINE共有ボタン
- X共有ボタン

結果画面では、通常表示用の画像と保存・共有用の画像を分けています。

- 画面表示: `result_driver_2.png`
- 保存用: `result_driver_2_share.png`

ローカルの `file://` で開いている間は、PC内のパスがLINE/Xの共有文に入らないようにしています。GitHub Pagesなどで `https://...` の公開URLから開かれた場合だけ、共有文に診断URLが自動で追加されます。
